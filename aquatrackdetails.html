🌊 AquaTrack: AI-Integrated Mobile App and Web System

AquaTrack is a capstone project designed to modernize the operations of water refilling stations in Butuan City. It combines a mobile and web-based system powered by Artificial Intelligence to recognize gallon containers, monitor inventory, and automate order management.

📘 Overview

AquaTrack enhances operational efficiency by integrating AI-powered detection through Python and TensorFlow, connected with real-time database updates in Supabase. The system was built using React and React Native for seamless cross-platform accessibility.

⚙️ Features

  • AI-based gallon container detection using TensorFlow and Flask API
  • Real-time data synchronization via Supabase
  • Responsive web dashboard for managing refilling operations
  • Mobile app for field monitoring and reporting
  • Automated inventory and order tracking

🧩 Technologies Used

React React Native Supabase Python TensorFlow Flask API HTML / CSS
